What I Did

  • Created a new email list in Mailchimp named "Test Registration List" for this demo.
  • Used Mailchimp's form builder to generate a clean, simple signup form with Email, First Name, and Last Name fields.
  • Embedded the Mailchimp form into a custom landing page with some styling.
  • Applied custom CSS for a modern, clean design that is fully responsive on mobile and desktop Since the Mailchimp free does not allow custom CSS I had to override the default styles.
  • Tested submissions to confirm emails correctly reach the Mailchimp list - all emails are collected and stored in the list.

This project showcases my ability to integrate external marketing tools, customize their appearance, and maintain full functionality and responsiveness within a web page. There are way more options for styling, cusgomization, validation, etc.
